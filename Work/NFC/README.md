# NFC

NFC (Near Field Communication，近場通訊) 是一種短距離、非接觸式的無線通訊技術，通常通訊範圍小於 10 公分。

NFC 是由 RFID (Radio Frequency Identification，無線射頻識別) 技術演進而來，主要設計目標是讓兩個裝置能夠快速、安全交換資料，並透過「靠近即可使用」的特性，大幅簡化使用流程。

常見支援 NFC 的裝置包括

- 智慧型手機

- 智慧手錶

- 悠遊卡、一卡通等非接觸式 IC 卡

- 門禁卡與門鎖

<br />

## NFC 的運作原理

NFC 運作於 13.56 MHz 頻段，資料傳輸速率介於 106 kbps 至 424 kbps。與一般無線通訊 (例如：Wi-Fi、藍牙) 不同，NFC 的特點是極短距離感應，這也是在安全性上的優勢，可以防止中間人攻擊 (Man-in-the-middle Attack，簡稱：MITM)。

NFC 主要有三種模式

- 讀取/寫入模式 (Reader/Writer Mode)

    - 手機或讀卡機可讀取或寫入具備 NFC 晶片的標籤 (Tag)。

    - 例如：用手機感應展場中的 NFC 標籤，即可開啟活動網站。

- 卡片模擬模式 (Card Emulation Mode)

    - 手機或裝置模擬成一張 IC 卡，讓感應式讀卡機能夠辨識。

    - 例如：Apple Pay、Google Pay、Samsung Pay 等行動支付。

- 點對點模式 (Peer-to-Peer Mode)

    - 兩個支援 NFC 的裝置可互相傳輸資料。

    - 例如：早期 Android 的「Android Beam」功能。

## NFC 的應用範例

- 行動支付與票證

    - Apple Pay、Google Pay、悠遊卡、一卡通、icash 等皆支援 NFC。

    - 使用者只需將手機或卡片靠近讀卡機即可完成付款或進出交通系統。

- 交通運輸

    - 大眾運輸 (捷運、公車、火車) 的電子票證系統幾乎都依賴 NFC 技術。

- 門禁系統與智慧鎖

    - 透過 NFC 卡或手機，就能開啟公司、社區大樓或飯店房間門鎖。

- 智慧生活與物聯網 (IoT)

    - NFC 標籤可用於智慧家庭場景，例如：將手機靠近 NFC 標籤，即可自動切換 Wi-Fi、開啟特定應用程式或控制家電。

- 身分識別與會員服務

    - NFC 常被應用於會員卡、電子票券或活動入場驗證。

<br />

## NFC 的優點

- 使用便利：不需開啟應用程式，只要靠近感應即可。

- 安全性高：因為感應距離極短，降低了被竊聽或中途攔截的風險。

- 相容性佳：大部分具備 NFC 功能的裝置可通用於不同場景。

- 耗電量低：與藍牙、Wi-Fi 相比，NFC 需要的能量極少。

<br />

## NFC 的限制與挑戰

- 通訊距離短：必須非常靠近才能使用，不適合大範圍資料傳輸。

- 傳輸速率有限：不適合用來傳送大檔案。

- 普及程度差異：雖然台灣的交通票證與支付已廣泛應用，但在部分國家或場景仍未全面普及。

- 裝置支援差異：不是所有智慧型手機都支援 NFC (部分低階機型可能沒有)。

<br />

## NFC 的未來發展

隨著 智慧支付、數位身分證、智慧城市 的推動，NFC 的應用將更加廣泛。未來可能的發展包括

- 整合電子身份認證 (例如：數位身分證或健保卡)

- 更智慧化的票券與會員管理系統

- 與 IoT 裝置深度結合，打造更便利的智慧生活

在台灣，隨著悠遊卡、一卡通與行動支付的推動，NFC 的角色只會越來越重要。

<br />

## 總結

NFC 是一項簡單、快速且安全的無線通訊技術，已經深深融入日常生活。從交通票證、行動支付、門禁到智慧家居，NFC 讓「感應一下」成為日常習慣。隨著數位化的浪潮持續推進，可以預見 NFC 將會是未來智慧城市與數位生活的重要基礎。
