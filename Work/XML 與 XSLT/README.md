# XML 與 XSLT

XML 與 XSLT 經常一起使用。XML 作為數據的儲存和傳輸格式，而 XSLT 則是用來將這些數據轉換為最終展示或處理的形式。這樣的組合特別適用於需要動態生成內容的系統，例如：基於 XML 的網頁應用，或需要在不同系統之間進行數據交換的情況。

<br />

## XML

XML，全名為「可擴展標記語言」 (eXtensible Markup Language)，是一種設計用來描述數據的標記語言。與 HTML 不同，HTML 的標籤集是固定的，主要用來控制網頁的展示方式，XML 的標籤是自定義的，專注於描述數據的結構與內容。因此，XML 並非用來「顯示」數據，而是用來「儲存」和「傳輸」數據。

- XML 的基本特徵

    - 自定義標籤：XML 允許用戶自定義標籤，根據數據的性質來命名標籤。與 HTML 不同，後者使用預定義的標籤 (例如：`<h1>`、`<p>` 等)。

    - 結構化數據：XML 使用嵌套的方式來表達數據的層次結構。這種層次結構非常適合描述父子關係的數據，例如：目錄結構、產品分類、或者組織架構等。

    - 平台無關性：XML 是一種純文本格式，因此可以在不同的平台和系統之間進行傳輸和交換。無論是 Windows、macOS 或 Linux，甚至是不同的程式語言 (例如：Java、Python、JavaScript)，都可以方便讀取和處理 XML 文件。

    - 語法嚴謹：XML 文件必須遵守特定的語法規則，例如：每個標籤必須正確開啟與關閉，並且必須保持正確的嵌套順序。這些嚴格的規則保證了 XML 文件的結構一致性，方便程式解析。

- 為什麼使用 XML

    XML 作為一種具有自定義標籤和結構化數據的語言，成為了解決跨平台數據傳輸的理想選擇。例如：在網路應用中，伺服器端可能會使用 XML 作為數據的傳輸格式，將資料傳送給用戶端，而用戶端則通過解析 XML 來展示或處理這些數據。

- XML 範例

    ```xml
    <書籍集>
      <書籍>
        <標題>誰是 Charmy</標題>
        <作者>Charmy</作者>
        <價格>1000000000000000</價格>
      </書籍>
      <書籍>
        <標題>Charmy 是誰</標題>
        <作者>Charmy</作者>
        <價格>1000000000000000</價格>
      </書籍>
    </書籍集>
    ```

    這段 XML 表示了一個書籍集，裡面包含兩本書的訊息。每本書籍都使用 `<書籍>` 標籤包裹，內部嵌套 `<標題>`、`<作者>` 和 `<價格>` 標籤，分別描述書籍的標題、作者以及價格。

## XSLT

XSLT，全名為「可擴展樣式表語言轉換」(eXtensible Stylesheet Language Transformations)，是一種專門用來轉換 XML 文件格式的技術。

- XSLT 的功能

    - 數據轉換：XSLT 的核心功能是將 XML 轉換為不同的格式。最常見的應用是將 XML 轉換為 HTML，從而將數據展示在網頁上。除此之外，XSLT 還可以將 XML 轉換為其他結構的 XML，或者純文本文件。

    - 數據過濾與重組：XSLT 不僅能夠轉換數據格式，還可以對數據進行篩選、過濾與重組。例如：可以使用 XSLT 來過濾掉某些不需要的數據，或者重新排列數據的順序，以符合特定需求。

    - 數據展示：XSLT 最常用於將數據轉換成 HTML，這樣瀏覽器就能夠讀取並展示這些數據，從而達到動態生成網頁的效果。

- XSLT 的工作原理

    XSLT 使用了一種稱為樣式表 (Stylesheet) 的文件，這個樣式表包含了一組模板，用來指導如何從 XML 中提取數據並進行格式化轉換。當 XSLT 處理 XML 文件時，會按照樣式表中的指示，逐步將 XML 文件中的節點轉換為所需的格式。

- XSLT 範例

    ```xml
    <?xml version="1.0" encoding="UTF-8"?>
    <xsl:stylesheet version="1.0" xmlns:xsl="http://www.w3.org/1999/XSL/Transform">
      <xsl:template match="/">
        <html>
          <body>
            <h2>書籍列表</h2>
            <ul>
              <xsl:for-each select="書籍集/書籍">
                <li>
                  <xsl:value-of select="標題"/> - 
                  <xsl:value-of select="作者"/> ($<xsl:value-of select="價格"/>)
                </li>
              </xsl:for-each>
            </ul>
          </body>
        </html>
      </xsl:template>
    </xsl:stylesheet>
    ```

    這個 XSLT 樣式表會將 XML 文件中的每本書籍轉換成 HTML 列表中的一個 `<li>` 項目，並顯示書籍的標題、作者和價格。

    `xsl:for-each>` 是 XSLT 用來迭代 XML 節點的語法，會遍歷 XML 文件中的每一個 `<書籍>` 節點，並將每本書的相關資料插入到 HTML 中。

    轉換後的 HTML 結果可能會如下

    ```html
    <html>
      <body>
        <h2>書籍列表</h2>
        <ul>
          <li>深入淺出 JavaScript - Charmy ($500)</li>
          <li>進階 Python 開發 - Charmy ($600)</li>
        </ul>
      </body>
    </html>
    ```

## 總結

- XML 是一種描述數據的標記語言，具有靈活性和可讀性，適用於多種應用場景。

- XSLT 是用來轉換 XML 的工具，可以將 XML 文件轉換為不同格式，例如：HTML、純文本或其他 XML 文件。
